NaFunny HUB 1.4.2 Telegram Engine Rewrite

Что изменено:
- Переписан Telegram parser engine.
- Добавлена жёсткая дедупликация по Telegram message id.
- Добавлена мягкая дедупликация похожих постов по началу очищенного текста.
- Парсер сортирует посты по Telegram message id, а не по порядку блоков в HTML.
- Улучшено извлечение картинок из Telegram HTML.
- PNG-эмодзи Telegram больше не считаются картинками постов.
- Добавлены подробные логи в GitHub Actions: scanned blocks / selected post ids.
- Cron обновления изменён на каждые 15 минут.
- Версия JSON обновлена до 1.4.2-telegram-engine-rewrite.

После загрузки:
1. Commit: NaFunny HUB 1.4.2 Telegram Engine Rewrite
2. Push origin
3. Actions → Update Telegram Feed → Run workflow
4. Проверить feed/telegram-feed.json и сайт.
