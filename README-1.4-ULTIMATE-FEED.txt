NaFunny HUB 1.4 Ultimate Feed

Что внутри:
- .github/workflows/update-telegram-feed.yml
- scripts/update-telegram-feed.mjs
- feed/telegram-feed.json
- telegram-feed.css
- telegram-feed.js

Как ставить:
1. Скопируй файлы и папки в корень репозитория через GitHub Desktop.
2. Commit: NaFunny HUB 1.4 Ultimate Feed
3. Push origin.
4. GitHub → Actions → Update Telegram Feed → Run workflow.
5. После зелёной галочки открой feed/telegram-feed.json и проверь posts.

Важно:
- @NaFunny и @TonNewbie должны быть публичными.
- Скрипт читает публичные страницы https://t.me/s/<channel>.
