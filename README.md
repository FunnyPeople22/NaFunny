# NaFunny Links Site

Готовый блок ссылок для GitHub Pages.

## Что внутри

- `index.html` — основная страница
- `styles.css` — стили
- `nafunny-links-all-in-one.html` — версия одним файлом, если удобнее всё держать в одном HTML

## Ссылки уже встроены

- YouTube
- Discord
- Instagram
- TikTok
- VK Group
- DonationAlerts / Support Me

## Как использовать

1. Залей `index.html` и `styles.css` в репозиторий GitHub Pages.
2. Если хочешь проще — используй только `nafunny-links-all-in-one.html`, переименуй его в `index.html`.
3. Включи GitHub Pages в настройках репозитория.
