# NaFunny Super Final

Финальная версия сайта NaFunny для GitHub Pages.

## Как загрузить

Загрузи все файлы из архива в корень репозитория:

- index.html
- style.css
- script.js
- logo.webp
- logo.png
- qr-trc20.png
- qr-ton.png
- qr-btc.png
- qr-eth.png
- README.md

Папок нет. Все пути в коде идут напрямую из корня репозитория.
